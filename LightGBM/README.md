# LightGBM
+ **Input -** Angle theta, Angle phi, Bending Angle, TimeInfo, RingNumber, Front/Rear hit, Mask, PatternStraightness, Zone, and MedianTheta
+ **Output -** 1/pT
+ **Loss -** MSE
+ **HyperParameters optimized -** Bagging fraction, Min child Sample, Max Depth, Lambda 1, Lambda 2, and Early Stopping
